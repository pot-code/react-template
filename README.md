# Features

- React-Query + Axios
- [Vite](https://vitejs.dev)
- [RTL(react-testing-library)](https://testing-library.com)
- Typescript
- Storybook
- [msw(Mock Service Worker)](https://mswjs.io/) API mocking

# Usage

## Setup

### Husky

```bash
npm i -D husky

# yarn
yarn add -D husky

# pnpm
pnpm add -D husky
```

### msw

```bash
npx msw init public/
```
